🎬 Movie Search App

📌 Project Overview

Movie Search App is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to search for movies and retrieve real-time information using the OMDb API. The application displays movie posters, ratings, release dates, genres, and plot summaries in an interactive and user-friendly interface.

🚀 Features
Search movies by title
Display movie poster
Show IMDb rating
Display release date
Show genre information
Display movie plot summary
Handle invalid search queries
Responsive user interface

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6+)
Fetch API
Async/Await
OMDb API
